# How to use this folder

This folder should hold all of the input data for your pipeline.
You can download the data [here](https://www.youtube.com/watch?v=dQw4w9WgXcQ).

Keep the folder structure from the download, which should include:
- Biomass
- Buildings
- Cargo
- Commodities
- Electricity_Transmission
- Energy_Conversion
- Gas
- geodata
- Hydro
- Industry
- Rdemand
- Transport
- VRE

The version_track.yml file is for keeping track of which version of data is used in a version of the pipeline. So check that the data you are using matches with the yml, otherwise the processing tools might have bugs.
